# Google Trends Monitor

End-to-end data pipeline to process data from [Google Trends](https://trends.google.com).

## Technologies
- Google Cloud Platform (GCP): Google Cloud Storage and BigQuery
- Docker
- Airflow
- dbt
- Python: pandas, pytrends
